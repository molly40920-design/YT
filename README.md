# 快速建立廣告 — 互動 Demo（XMP 移植）

參考 XMP（mobvista）的「一鍵交叉建立廣告」功能，移植於自有平台的互動原型。
三欄式建立工作區：左欄 9 大步驟導航、中欄動態表單（連動防呆）、右欄即時拆分規則。

## 線上檢視

這是一個**單一檔案、零安裝**的原型，所有相依套件（React、Babel、Tailwind）皆由 CDN 載入，圖示為內嵌 SVG。

兩種開啟方式：

1. **直接下載開啟**：下載 `index.html`，用瀏覽器打開即可（需連網以載入 CDN）。
2. **GitHub Pages**：將本 repo 推上 GitHub → Settings → Pages → 選擇 branch（如 `main`）/ root → 儲存，稍候即可用 `https://<帳號>.github.io/<repo>/` 開啟。

> 註：GitHub 直接預覽 `.html` 只會顯示原始碼；要看實際畫面請用上述任一方式於瀏覽器執行。

## 功能重點

- **四個多組合模組**：地區組、定向包、出價和預算、創意組，可用分頁「新增」（上限 30 / 20 / 20 / 30），右欄「拆分規則」表即時相乘。
- **關鍵連動**：iOS 14+ 開關（鎖 App Store、改變成效目標與出價策略名稱）、CBO 開關（預算與出價在 Campaign 層或廣告組層互換）、成效目標 × 出價策略過濾、歸因視窗連動。
- **創意組**：創意類型／格式可複選，各格式各自展開素材區並加總廣告數。
- **預覽**：點「預覽」列出即將生成的每個廣告組（狀態可切換），底部可發布。

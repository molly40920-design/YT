隱私權政策 (Privacy Policy)

最後更新日期：2026年6月9日

本隱私權政策旨在說明「智能營銷：YT上傳系統」（以下簡稱本系統）如何收集、使用、儲存與保護您的資料。本系統為鈊象電子 (IGS) 內部專用之營運輔助工具。

1. 第三方服務與 API 規範遵守

本系統之核心影音上傳功能，係透過存取 YouTube API 服務 (YouTube API Services) 來實現。因此，當您使用本系統進行驗證與授權時，即表示您已詳細閱讀並同意受下列條款約束：

  • YouTube 服務條款 (YouTube Terms of Service)
  • Google 隱私權政策 (Google Privacy Policy)

2. 我們收集與使用的資料範圍

為了確保自動化影片上傳作業順利執行，並驗證內部人員身分，我們僅會在您明確點擊 Google OAuth 授權後，存取以下最低限度的資訊：

  • Google 帳號基本資訊： 包含您的電子郵件地址，僅用於驗證您是否具備 IGS 內部系統的存取權限。
  • YouTube 頻道存取權限： 系統需獲取管理頻道影片的權限，以便代表您自動將行銷影片上傳至指定的官方頻道，並設定標題、說明欄、隱私狀態等影片中介資料 (Metadata)。

特別聲明：本系統「不會」讀取您的私人觀看紀錄、「不會」收集與工作無關之個人資訊，也「不會」追蹤您在其他網站的瀏覽行為。

3. 資料儲存政策與安全性控制

我們採用業界標準的加密技術來保護您的授權資訊，並嚴格執行「資料不落地」原則：

  • 憑證管理： 您的授權 Token 採加密方式安全儲存於封閉的內部伺服器，且該 Token 僅專供自動化上傳之 Cron Job 任務讀取使用。
  • 資料刪除與更新： 我們不會永久儲存由 YouTube API 回傳的歷史數據。所有於上傳過程中快取之狀態資料（如上傳進度、報表數據）皆設定有存活時間 (TTL)，並會於 24 小時內自動覆寫或永久清除。

4. 資料分享與揭露限制

本系統收集之所有授權資料均嚴格限制於鈊象 (IGS) 內部系統執行層面使用。我們承諾，絕對不會將您的個人資訊、頻道資料或上傳之內容分享、出售、出租或轉讓給任何外部第三方機構或廣告商。

🛡️ 撤銷存取權限與資料刪除 (Revoking Access)

您擁有對自己 Google 帳號授權的絕對控制權。您可以隨時取消本系統對您 Google 帳號的存取權：

  1. 請前往您的 Google 帳戶安全性設定頁面 (Google Security Settings)。
  2. 在「具備帳戶存取權的第三方應用程式」清單中，找到「智能營銷：YT上傳」。
  3. 點擊「移除存取權」。

當您執行撤銷後，本系統會立即偵測到權限終止，並同步將我們資料庫中與您關聯的加密 Token 與暫存資料永久刪除，停止所有相關的排程任務。

# 快速建立廣告 — 互動 Demo（XMP 移植）

> 本 `index.html` 為**完全自包含**版本：React 與樣式（Tailwind）已預先打包並內嵌，**不需連外、不需 CDN、不需瀏覽器即時編譯**。因此可在 GitHub Pages、公司內網或離線環境直接開啟，秒載入。

## 線上檢視 / 部署

1. **直接開啟**：下載 `index.html`，用瀏覽器打開即可（離線也可）。
2. **GitHub Pages**：把 `index.html` 放到 repo（或 `/YT/` 子資料夾）→ Settings → Pages → 選 branch / root → 儲存。網址例：`https://<帳號>.github.io/<repo>/`（子資料夾則加 `/YT/`）。更新檔案後請用 Ctrl+F5 強制重新整理（避免快取舊版）。

> 若先前版本卡在「載入中…」：那是舊版依賴外部 CDN（unpkg／cdnjs／tailwind）與瀏覽器 Babel 即時編譯，在內網被擋或編譯過慢所致。本版已移除這些相依，覆蓋上去即可。

## 平台切換

於 **基礎設置** 的「投放平台」欄位選擇 META 或 Google 即切換對應流程（TikTok／ASA／Applovin 後續實作）。

- **Meta（9 步）**：基礎設置 → 廣告系列 → 投放內容 → 地區組 → 版位 → 定向包 → 出價和預算 → 創意設置 → 創意組。四個多組合模組（地區組／定向包/出價和預算/創意組）皆拆分於「廣告組」層。iOS 14+、CBO、成效目標×出價策略等連動完整。
- **Google UAC（5 步）**：基礎設置 → 地區組 → 出價和預算 → 廣告組 → 創意組。兩個多組合模組：地區組（拆分於「廣告系列」層，上限 100）、創意組（拆分於「廣告組」層，上限 50）。成效目標（安裝量／應用內操作次數／應用內操作價值 tROAS）連動出價與欄位。

## 預覽

點「預覽」列出即將生成的每個廣告組（狀態可切換）；「操作」可**複製**（下方插入 `_copy` 列）與**刪除**，摘要數字即時更新。

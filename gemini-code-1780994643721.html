智能營銷：YT上傳系統

鈊象 (IGS) 內部專用影音排程與上傳管理中樞

系統概述

「智能營銷：YT上傳系統」為鈊象電子 (IGS) 內部營運與行銷團隊專屬開發之自動化影音管理工具。本系統旨在簡化並自動化 YouTube 頻道的影片排程與發布流程，協助行銷團隊更有效率地管理多個官方頻道的影音資產，確保數位行銷策略能精準落地。

核心功能特色

🚀 自動化上傳與排程

支援批次處理行銷影片，可直接於系統內預先設定 YouTube 發布時間、標題與影片說明，減少人工作業時間。

📊 多頻道集中管理

提供單一操作介面，允許內部授權人員快速切換並管理旗下多款遊戲的獨立 YouTube 頻道狀態。

🔒 嚴格的權限控管

系統架構與內部員工權限深度綁定，確保僅有獲得授權的特定行銷專員與營運人員可執行寫入與上傳作業。

📈 狀態追蹤日誌

完整記錄每一筆影片上傳的狀態回報與 API 響應結果，便於團隊追蹤發布進度與除錯。
